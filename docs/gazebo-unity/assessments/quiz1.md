# Quiz 1: Gazebo/Unity Simulation Concepts

## Instructions

This quiz assesses your understanding of simulation concepts using Gazebo and Unity. Choose the best answer for each question. You may refer to the course materials during this open-book quiz.

## Questions

A) To completely replace real-world testing
B) To provide a safe, risk-free environment for testing robot behaviors
C) To make robots unnecessary
D) To reduce the need for programming

### Question 2: Which of the following is a key advantage of Gazebo for robotics simulation?

A) Photo-realistic rendering
B) Physics-accurate simulation with realistic collision detection
C) Easy game development
D) Better graphics performance than Unity

### Question 3: In Gazebo, what does SDF stand for?

A) Simulation Description Format
B) Sensor Data Framework
C) Standard Dynamics Format
D) System Definition File

B) ML-Agents
C) Unity Perception
D) Robotics SDK

A) Physics simulation only
B) High-fidelity rendering for perception tasks
C) Hardware interface only
D) Network communication

### Question 6: In Gazebo, how do you typically interface with ROS 2?

A) Through direct function calls
B) Using Gazebo plugins
C) Via shared memory
D) Through file exchange

### Question 7: What is a "digital twin" in the context of robotics?

A) A duplicate robot in the real world
B) A virtual replica of a physical system used for simulation and analysis
C) A type of sensor
D) A programming framework

### Question 8: Which sensor model in Unity provides realistic camera simulation?

A) CameraSensor
B) PerceptionCamera
C) UnityCamera
D) SimCamera

### Question 9: What is the typical workflow for validating robot behaviors using simulation?

B) Only test on real robots

B) Differences between simulated and real-world physics, sensors, and environments
C) Simulation requires too much computational power
D) Robots cannot run in simulation
