# Quiz 1: ROS 2 Architecture Fundamentals

## Instructions

This quiz assesses your understanding of ROS 2 architecture fundamentals covered in Weeks 1-2. Choose the best answer for each question. You may refer to the course materials during this open-book quiz.

## Questions

A) Data Definition System
B) Data Distribution Service
C) Dynamic Discovery System
D) Distributed Data Service

### Question 2: Which of the following is NOT a basic communication primitive in ROS 2?

A) Topics
B) Services
C) Parameters
D) Databases

A) To store data persistently
B) To provide a graphical user interface
C) The basic unit of execution that performs computation

A) A system where nodes request data from a server
B) A one-to-many communication pattern where one node sends messages to many listeners
