# Gazebo/Unity کوئز 1

## نامزدگان کے لیے تقاضے

اس کوئز کو گزرنے کے لیے، آپ کو درج ذیل موضوعات کو سمجھنا چاہیے:

- Gazebo کی بنیادیں
- Unity سیمولیشن کے فوائد
- سیمولیشن ماحول کو تیار کرنا
- سینسر سیمولیشن
- ROS 2 کے ساتھ انضمام

## سوالات

1. **Gazebo میں کون سا فزکس انجن استعمال کیا جاتا ہے؟**
   - A) PhysX
   - B) Bullet
   - C) ODE
   - D) B اور C دونوں

2. **Unity کا کون سا فائدہ روبوٹکس سیمولیشن کے لیے خاص طور پر اہم ہے؟**
   - A) بہتر فزکس سیمولیشن
   - B) فوٹو ریلزم رینڈرنگ
   - C) کم کمپیوٹیشنل ضروریات
   - D) آف لائن کام

3. **ROS 2 کو Gazebo سے کیسے منسلک کیا جاتا ہے؟**
   - A) Gazebo-Ros پلگ انز کے ذریعے
   - B) براہ راست کوڈ کنکشن
   - C) ایپی آئیز کے ذریعے
   - D) HTTP کالز کے ذریعے

4. **سیمولیشن سے حقیقت میں منتقلی کے چیلنج کو کیا کہا جاتا ہے؟**
   - A) Reality gap
   - B) Sim-to-real gap
   - C) Transfer problem
   - D) All of the above

5. **کون سا Unity اوزار ROS سے مربوط ہونے کے لیے استعمال کیا جاتا ہے؟**
   - A) Unity ROS Bridge
   - B) ROS# 
   - C) Unity Robotics Package
   - D) تمام مندرجہ بالا

## جوابات

1. D - Gazebo ODE، Bullet، یا DART کے طور پر متعدد فزکس انجن سپورٹ کر سکتا ہے
2. B - Unity کا فوٹو ریلزم رینڈرنگ اسے تاثرات کی تربیت کے لیے بہتر بناتا ہے
3. A - Gazebo-Ros پلگ انز ROS 2 کنکشن کے لیے استعمال کیے جاتے ہیں
4. D - تمام اصطلاحات ایک ہی مسئلے کو بیان کرتی ہیں
5. D - تمام اوزار ROS کنکشن کے لیے استعمال کیے جا سکتے ہیں
