# NVIDIA Isaac Assessment 1

## تفصیل

یہ کوئز perception systems اور VSLAM کے بارے میں آپ کی سمجھ کو جانچے گا۔

## سوالات

1. **NVIDIA Isaac کا کون سا پیکج AI inference کے لیے استعمال ہوتا ہے؟**
   - A) Isaac ROS Vision
   - B) Isaac ROS DNN Image Encoding
   - C) Isaac ROS Camera
   - D) Isaac ROS Sensors

2. **VSLAM کا مطلب کیا ہے؟**
   - A) Visual Spatial Localization and Mapping
   - B) Video Simultaneous Localization and Mapping
   - C) Visual Simultaneous Localization and Mapping
   - D) Virtual Spatial Localization and Mapping

3. **NVIDIA Jetson پلیٹ فارم کا کیا فائدہ ہے؟**
   - A) کم بجلی استعمال
   - B) AI inference کے لیے GPU تیزی
   - C) کم کمپیوٹیشنل فوٹ پرنٹ
   - D) تمام مذکورہ

4. **Isaac Lab کے استعمال کا کیا مقصد ہے؟**
   - A) سیمولیشن میں روبوٹکس الگورتھم کی تربیت
   - B) Hardware debugging
   - C) Network configuration
   - D) فقط 3D رینڈرنگ

5. **TensorRT کا استعمال کیوں کیا جاتا ہے؟**
   - A) کم memory استعمال
   - B) AI model inference کی کارکردگی کو بہتر بنانا
   - C) فقط 3D گریفکس
   - D) Hardware compatibility

## جوابات

1. B - Isaac ROS DNN Image Encoding
2. C - Visual Simultaneous Localization and Mapping
3. D - تمام مذکورہ
4. A - سیمولیشن میں روبوٹکس الگورتھم کی تربیت
5. B - AI model inference کی کارکردگی کو بہتر بنانا
