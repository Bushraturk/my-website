# VLA Assignment 1

## تفصیل

اس تفویض کا مقصد Vision-Language-Action pipeline تیار کرنا ہے جو natural language کمانڈز کو perception اور action planning میں تبدیل کرتا ہے۔

## تقاضے

- Ubuntu 20.04 اور ROS 2 Humble
- NVIDIA GPU یا Jetson Platform
- Isaac ROS packages
- GPT یا open-source LLM

## ہدایات

### Part 1: Vision-Language Integration (20 نمبر)

1. Camera feed کو AI perception system سے منسلک کریں
2. Object detection اور semantic segmentation کو نافذ کریں
3. Vision processing کے نتائج کو language model کے ساتھ ضم کریں
4. Results کو visualizer میں display کریں

### Part 2: Action Planning (15 نمبر)

1. Language command کے مطابق action plan تیار کریں
2. Plan کو robot control stack میں convert کریں
3. Safety checks اور validation شامل کریں
4. Action execution pipeline تیار کریں

### Part 3: VLA Integration (15 نمبر)

1. Vision-Language-Action pipeline کو کمپلیٹ طور پر ضم کریں
2. End-to-end system test کریں
3. Performance metrics record کریں
4. System کی robustness کو validate کریں

### Part 4: Documentation (5 نمبر)

1. System architecture کی تفصیل فراہم کریں
2. Performance analysis شامل کریں
3. چیلنج اور ان کے حل بیان کریں

## گریڈنگ کریٹیریا

- **Vision-Language Integration**: 20 نمبر - 100% functionality کے ساتھ
- **Action Planning**: 15 نمبر - Complete implementation
- **VLA Integration**: 15 نمبر - Proper end-to-end functionality
- **Documentation**: 5 نمبر - Comprehensive and clear documentation

## جمع کروانے کے لیے تقاضے

1. GitHub ریپوزٹری پر اپنے کوڈ push کریں
2. System architecture کی تفصیل کے ساتھ رپورٹ جمع کرائیں
3. Performance metrics record کریں
4. Testing results document کریں
