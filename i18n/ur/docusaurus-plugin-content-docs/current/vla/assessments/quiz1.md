# VLA Assessment 1

## تفصیل

یہ کوئز Vision-Language-Action ماڈلز کے بارے میں آپ کی سمجھ کو جانچے گا۔

## سوالات

1. **VLA کا مطلب کیا ہے؟**
   - A) Vision-Language-Action
   - B) Vision-Language-Agent
   - C) Visual-Linguistic-Actuation
   - D) Vector-Language-Algorithm

2. **VLA models کا کیا مقصد ہے؟**
   - A) فقط images کو سمجھنا
   - B) فقط text کو process کرنا
   - C) Vision، language، اور action کو end-to-end میں ضم کرنا
   - D) فقط speech کو recognize کرنا

3. **Embodied AI کیا ہے؟**
   - A) AI models کو physical bodies سے مربوط کرنا
   - B) AI models کو cloud میں run کرنا
   - C) AI models کو کمپیوٹر میں استعمال کرنا
   - D) AI models کو صرف text کے لیے استعمال کرنا

4. **VLA systems کا استعمال کیا کرتا ہے؟**
   - A) perception، reasoning، اور action planning
   - B) فقط data storage
   - C) فقط network communication
   - D) فقط hardware control

5. **Vision-Language-Action models میں کون سی چیزیں شامل ہوتی ہیں؟**
   - A) Computer vision
   - B) Natural language processing
   - C) Robot control
   - D) تمام مذکورہ

## جوابات

1. A - Vision-Language-Action
2. C - Vision، language، اور action کو end-to-end میں ضم کرنا
3. A - AI models کو physical bodies سے مربوط کرنا
4. A - perception، reasoning، اور action planning
5. D - تمام مذکورہ
