# ترجمہ کیسے شامل کریں گائیڈ

## خلاصہ

یہ دستاویزات سائٹ میں اردو ترجمہ کی حمایت کو شامل کرنے کا طریقہ بیان کرتی ہے۔

## ضروریات

- ڈوکوسرس کا بنیادی علم
- ڈوکوسرس کنفیگریشن کی فائل کے بارے میں علم
- ٹرانس لیشن فائل کی سٹرکچر کے بارے میں علم

## ترجمہ شامل کریں

### 1. ڈوکوسرس کنفیگریشن کو اپ ڈیٹ کریں

`docusaurus.config.js` فائل میں، i18n کنفیگریشن کو شامل کریں:

```js
module.exports = {
  // ... دیگر کنفیگریشن
  i18n: {
    defaultLocale: 'en',
    locales: ['en', 'ur'],
    localeConfigs: {
      en: {
        label: 'English',
        direction: 'ltr',
      },
      ur: {
        label: 'اردو',
        direction: 'rtl',
      },
    },
  },
};
```

### 2. ٹرانس لیشن فائلیں تیار کریں

`i18n/ur/docusaurus-plugin-content-docs/current/` ڈائریکٹری میں، ہر انگریزی فائل کا مطابق اردو ورژن تیار کریں:

- ہر فائل میں فرنٹ میٹر کو شامل کریں
- میٹا ڈیٹا کو تبدیل کیے بغیر، صرف مواد کا ترجمہ کریں

### 3. نیوی گیشن کو اپ ڈیٹ کریں

`sidebars.js` فائل میں، دونوں زبانوں کے لیے فائل کے آئی ڈی ایک ہی رکھیں تاکہ ڈوکوسرس دونوں زبانوں کے درمیان اچھی طرح سے سوئچ کرے:

```js
module.exports = {
  textbookSidebar: [
    {
      type: 'category',
      label: 'متعارف',
      items: [
        'intro',  // یہ آئی ڈی دونوں زبانوں میں ایک ہی رہنی چاہیے
        'home-test',
        // ... دیگر اشیاء
      ],
    },
  ],
};
```

## ٹرانس لیشن کے لیے بہترین طریقے

- میٹا ڈیٹا (عنوان، سائڈ بار پوزیشن، وغیرہ) کو تبدیل نہ کریں
- دونوں زبانوں میں فائل کی آئی ڈی ایک ہی رکھیں
- سائڈ بارز کے ڈھانچے کو دونوں زبانوں میں ایک جیسا رکھیں
- ٹیکنیکل اصطلاحات کو اردو میں ترجمہ کریں یا ان کو انگریزی میں چھوڑ دیں اور قوسین میں اردو ترجمہ شامل کریں

## ٹیسٹنگ

- ہر ترجمہ کی فائل کو جانچیں کہ وہ ڈوکوسرس سائٹ میں ٹھیک سے لوڈ ہو رہی ہے
- دونوں زبانوں کے درمیان سوئچ کریں اور یقین کریں کہ نیوی گیشن کام کر رہی ہے
- ٹرانس لیشن فائلز کو دوبارہ بنایں اور ڈیپلائی کریں
