# دستاویزات کے ورژنز کا انتظام

Docusaurus آپ کی دستاویزات کے متعدد ورژنز کو منظم کر سکتا ہے۔

## نیا ورژن بنائیں

```bash
npm run docusaurus docs:version 1.0
```

## ورژن کی ترتیب

`docusaurus.config.js` میں ورژنز کی ترتیب دی جا سکتی ہے۔
