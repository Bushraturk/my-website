# اپنی سائٹ کا ترجمہ کریں

Docusaurus i18n فیچر کا استعمال کرتے ہوئے اپنی سائٹ کا ترجمہ کریں۔

## ترجمے کی فائلز بنائیں

```bash
npm run write-translations -- --locale ur
```

## ترجمے کی ترتیب

`docusaurus.config.js` میں زبانوں کی ترتیب دیں۔
