# ٹیوٹوریل کے اضافی جز (Tutorial Extras)

یہ اضافی ٹیوٹوریلز فزکل AI اور ہیومنوائڈ روبوٹکس کورس سے باہر تکنیکی وضاحتیں اور اہم تکنیکی تجاویز فراہم کرتے ہیں۔

یہ ٹیوٹوریلز درج ذیل موضوعات پر مشتمل ہیں:

- مستند کرنا (Authentication)
- ڈیٹا کی تجدید (Data updates)
- ٹیکسٹ بک کا ترجمہ (Textbook translation)
- ڈیبگنگ کی سرگرمیاں (Debugging activities)
