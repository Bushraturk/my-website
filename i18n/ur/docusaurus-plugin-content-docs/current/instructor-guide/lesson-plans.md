# انسٹرکٹر گائیڈ - ہفتہ وار سبق کے منصوبے

یہ سیکشن فزیکل AI اور ہیومنوائڈ روبوٹکس کورس کے ہر ہفتے کے لیے تفصیلی سبق کے منصوبے فراہم کرتا ہے۔

## عمومی ہفتہ وار ڈھانچہ

ہر ہفتہ عام طور پر اس ڈھانچے کی پیروی کرتا ہے:
- **دن 1**: تھیوری اور تصورات کا تعارف
- **دن 2**: Implementation اور hands-on مشق
- **دن 3**: لیب مشقیں اور پروجیکٹ کا کام
- **دن 4**: جائزہ اور troubleshooting
- **دن 5**: پروجیکٹ کا کام اور ایڈوانسڈ موضوعات

## ہفتہ 1-2: ROS 2 آرکیٹیکچر اور تصورات

### سیکھنے کے مقاصد
- طلباء ROS 2 کی بنیادی آرکیٹیکچر سمجھیں گے
- طلباء Python میں بنیادی ROS 2 nodes بنائیں اور چلائیں گے
- طلباء topics اور services استعمال کرتے ہوئے nodes کے درمیان message passing لاگو کریں گے

### دن 1: ROS 2 آرکیٹیکچر کا جائزہ
**مدت**: 2 گھنٹے لیکچر + 1 گھنٹہ لیب تیاری

**سرگرمیاں**:
1. **لیکچر** (60 منٹ):
   - ROS 2 بمقابلہ ROS 1 آرکیٹیکچر کے فرق
   - DDS (Data Distribution Service) تصور
   - Nodes, topics, services, actions کی وضاحت

2. **لیب سیٹ اپ** (30 منٹ):
   - طلباء ورک سٹیشنز پر ROS 2 انسٹالیشن کی تصدیق کریں
   - بنیادی ROS 2 commands کا مظاہرہ

### دن 2: ROS 2 Nodes بنانا
**مدت**: 1 گھنٹہ لیکچر + 3 گھنٹے لیب

**سرگرمیاں**:
1. **لیکچر** (30 منٹ):
   - Node ساخت اور lifecycle
   - Publisher اور subscriber implementation

2. **Hands-on مشق** (2.5 گھنٹے):
   - طلباء اپنا پہلا ROS 2 node بنائیں
   - Publisher اور subscriber nodes لاگو کریں

### دن 3: Services اور Actions
**مدت**: 1 گھنٹہ لیکچر + 3 گھنٹے لیب

**سرگرمیاں**:
- Services بمقابلہ topics: ہر ایک کب استعمال کریں
- طویل چلنے والے tasks کے لیے Actions
- Error handling اور node resilience

## ہفتہ 4-5: Gazebo Simulation Environments

### سیکھنے کے مقاصد
- طلباء Gazebo میں simulation environments بنائیں گے
- طلباء simulation میں روبوٹ ماڈلز اور sensors لاگو کریں گے

## ہفتہ 7-8: NVIDIA Isaac کے ساتھ Perception اور VSLAM

### سیکھنے کے مقاصد
- طلباء Isaac tools استعمال کرتے ہوئے perception algorithms لاگو کریں گے
- طلباء روبوٹ navigation کے لیے VSLAM سسٹم بنائیں گے

## ہفتہ 10-11: Vision-Language Integration

### سیکھنے کے مقاصد
- طلباء vision models کو language understanding سے جوڑیں گے
- طلباء multimodal fusion تکنیکیں لاگو کریں گے

## ہفتہ 13: کورس کی ترکیب اور Capstone پروجیکٹ

### سیکھنے کے مقاصد
- طلباء تمام کورس ماڈیولز کو ایک سسٹم میں ضم کریں گے
- طلباء جامع embodied AI سسٹم کا مظاہرہ کریں گے

## جائزہ کی حکمت عملیاں

### Formative Assessment Tools
- لیکچرز کے دوران Concept checks
- Peer programming مشقیں
- Real-time debugging challenges
- Code review sessions

### Summative Assessment حکمت عملیاں
- تکنیکی تجزیہ کے ساتھ ہفتہ وار لیب رپورٹس
- Rubric-based grading کے ساتھ programming assignments
- مظاہرے کے ساتھ پروجیکٹ پریزنٹیشنز
- جامع فائنل پروجیکٹ کا جائزہ

## عام طلباء کی غلط فہمیاں

### ROS 2 مخصوص غلط فہمیاں
- Asynchronous بمقابلہ synchronous communication میں الجھن
- Node lifecycle اور error handling کی غلط سمجھ
- Topics, services, اور actions کے استعمال کے منظرناموں میں الجھن

### Simulation مخصوص غلط فہمیاں
- یہ فرض کرنا کہ simulation حقیقت سے بالکل میل کھاتی ہے
- Simulation میں computational complexity کو نظرانداز کرنا
