# انسٹرکٹر گائیڈ - ہارڈویئر سیٹ اپ

یہ سیکشن فزیکل AI اور ہیومنوائڈ روبوٹکس کورس کے لیے ہارڈویئر انفراسٹرکچر سیٹ اپ کی جامع ہدایات فراہم کرتا ہے۔

## لیب سٹیشن کی ضروریات

### کم از کم تفصیلات (فی سٹوڈنٹ سٹیشن)
- **کمپیوٹر**: Intel i7-10700K یا AMD Ryzen 7 3700X
- **RAM**: 32GB DDR4
- **GPU**: NVIDIA RTX 3070 (8GB VRAM) یا بہتر
- **سٹوریج**: 500GB NVMe SSD
- **نیٹ ورک**: Gigabit Ethernet
- **OS**: Ubuntu 22.04 LTS

### تجویز کردہ تفصیلات
- **کمپیوٹر**: Intel i9-12900K یا AMD Ryzen 9 5900X
- **RAM**: 64GB DDR4
- **GPU**: NVIDIA RTX 4080/4090 (16GB+ VRAM)
- **سٹوریج**: 1TB NVMe SSD + 2TB HDD

## روبوٹ پلیٹ فارم سیٹ اپ

### Universal Robot (UR3/UR5)
1. **ابتدائی سیٹ اپ**:
   - URScript development environment انسٹال کریں
   - ورک سٹیشن اور روبوٹ کے درمیان نیٹ ورک کنکشن
   - مناسب safety zones کے ساتھ safety configuration

2. **سافٹ ویئر انٹیگریشن**:
   - ROS 2 UR driver packages انسٹال کریں
   - مخصوص روبوٹ ماڈل کے لیے MoveIt! planning configure کریں

### Franka Emika Panda
1. **ابتدائی سیٹ اپ**:
   - اپنی فزیکل لوکیشن میں روبوٹ کیلیبریٹ کریں
   - مناسب control modes کے ساتھ FCI انسٹال کریں

2. **سافٹ ویئر انٹیگریشن**:
   - franka_ros2 packages انسٹال کریں
   - کیمرہ-ٹو-روبوٹ hand-eye coordination کیلیبریٹ کریں

## ویژن سسٹم سیٹ اپ

### Intel RealSense D435/D435i
1. **ہارڈویئر انسٹالیشن**:
   - workspace کے لیے مناسب field of view کے ساتھ کیمرہ mount کریں
   - USB 3.0+ port سے کنیکٹ کریں

2. **سافٹ ویئر کنفیگریشن**:
   - RealSense ROS 2 wrapper انسٹال کریں
   - کیمرہ intrinsic اور extrinsic parameters کیلیبریٹ کریں

## نیٹ ورک اور انفراسٹرکچر

### لوکل نیٹ ورک کنفیگریشن
- روبوٹ communication کے لیے dedicated subnet
- روبوٹ controllers کے لیے static IP configuration
- روبوٹ communication کو ترجیح دینے کے لیے QoS configuration

## سیفٹی ایکوپمنٹ

### لازمی سیفٹی آئٹمز
- **سیفٹی گوگلز**: روبوٹکس لیب ماحول کے لیے مناسب
- **فرسٹ ایڈ کٹ**: آسانی سے قابل رسائی
- **فائر ایکسٹنگوشر**: الیکٹریکل ایکوپمنٹ کے لیے مناسب کلاس
- **ایمرجنسی کانٹیکٹس**: پوسٹ شدہ

## دیکھ بھال کا شیڈول

### روزانہ چیک (ہر لیب سیشن سے پہلے)
- روبوٹ homing اور بنیادی فعالیت کی تصدیق کریں
- تمام ڈیوائسز کی نیٹ ورک کنیکٹیوٹی چیک کریں
- سیفٹی سسٹمز کام کر رہے ہیں یقینی بنائیں

### ہفتہ وار دیکھ بھال
- سافٹ ویئر پیکجز اور سیکیورٹی پیچز اپڈیٹ کریں
- روبوٹ کیلیبریشن چیک کریں
- ویژن سسٹم لینز صاف کریں

### ماہانہ دیکھ بھال
- تمام سسٹم configurations کا جامع بیک اپ
- گھسے ہوئے اجزاء چیک اور تبدیل کریں
- سیفٹی طریقہ کار کا جائزہ لیں

## عام مسائل کا حل

### نیٹ ورک کنیکٹیوٹی مسائل
- **علامت**: روبوٹ کمانڈز کا جواب نہیں دے رہا
- **حل**: IP configuration، firewall settings، اور کیبل کنیکشنز چیک کریں

### ویژن سسٹم کیلیبریشن ڈرفٹ
- **علامت**: object detection یا positioning میں کم ہوتی درستگی
- **حل**: کیمرے دوبارہ کیلیبریٹ کریں اور mounting stability کی تصدیق کریں
