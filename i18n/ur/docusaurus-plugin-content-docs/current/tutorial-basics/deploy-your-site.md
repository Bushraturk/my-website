# اپنی سائٹ کو تعینات کریں

Docusaurus ایک **static-site-generator** ہے (جسے **[Jamstack](https://jamstack.org/)** بھی کہا جاتا ہے)۔

یہ آپ کی سائٹ کو سادہ **static HTML, JavaScript اور CSS فائلز** کے طور پر بناتا ہے۔

## اپنی سائٹ بنائیں

اپنی سائٹ کو **production کے لیے** بنائیں:

```bash
npm run build
```

Static فائلز `build` فولڈر میں پیدا ہوتی ہیں۔

## اپنی سائٹ کو تعینات کریں

اپنے production build کو locally ٹیسٹ کریں:

```bash
npm run serve
```

اب `build` فولڈر [http://localhost:3000/](http://localhost:3000/) پر دستیاب ہے۔

آپ اب `build` فولڈر کو **تقریباً کہیں بھی** آسانی سے تعینات کر سکتے ہیں، **مفت** یا بہت کم قیمت پر (مزید پڑھیں **[Deployment Guide](https://docusaurus.io/docs/deployment)**)۔
