# مبارک ہو!

آپ نے ابھی **Docusaurus کی بنیادیں** سیکھ لی ہیں اور **ابتدائی template** میں کچھ تبدیلیاں کی ہیں۔

Docusaurus میں **اور بھی بہت کچھ ہے**!

کیا آپ کے پاس **5 مزید منٹ** ہیں؟ **[versioning](../tutorial-extras/manage-docs-versions.md)** اور **[i18n](../tutorial-extras/translate-your-site.md)** پر نظر ڈالیں۔

اس tutorial میں کچھ **غیر واضح** یا **buggy** ہے؟ [براہ کرم رپورٹ کریں!](https://github.com/facebook/docusaurus/discussions/4610)

## آگے کیا؟

- [سرکاری documentation](https://docusaurus.io/) پڑھیں
- [`docusaurus.config.js`](https://docusaurus.io/docs/api/docusaurus-config) کے ساتھ اپنی سائٹ configuration میں ترمیم کریں
- [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) کے ساتھ navbar اور footer items شامل کریں
- ایک custom [Design and Layout](https://docusaurus.io/docs/styling-layout) شامل کریں
- ایک [search bar](https://docusaurus.io/docs/search) شامل کریں
- [Docusaurus showcase](https://docusaurus.io/showcase) میں inspirations تلاش کریں
- [Docusaurus Community](https://docusaurus.io/community/support) میں شامل ہوں
