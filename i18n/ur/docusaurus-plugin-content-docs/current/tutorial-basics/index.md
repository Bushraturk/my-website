# ٹیوٹوریل کی بنیاد (Tutorial Basics)

یہ ٹیوٹوریلز آپ کو فزکل AI اور ہیومنوائڈ روبوٹکس کورس میں استعمال ہونے والی بنیادی تکنیکی ترتیبات اور مفادات کو سمجھنے میں مدد دیں گے۔

یہ ٹیوٹوریلز درج ذیل موضوعات پر مشتمل ہیں:

- Docusaurus framework کا استعمال کرنا
- ROS 2 بنیادیات
- MDX components کا استعمال کرنا
- ٹیکسٹ بک کی نیوی گیشن
