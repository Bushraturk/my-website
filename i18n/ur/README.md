# فزیکل AI اور ہیومنوائڈ روبوٹکس کی کتاب (Urdu Translation)

## تعارف

یہ کتاب 13 ہفتے کا جامع کورس ہے جو جدید AI اور روبوٹکس کے تقاطع کو کور کرتا ہے۔ اس کتاب میں، آپ فزیکل AI سسٹم تیار کریں گے جو ROS 2، NVIDIA Isaac، Gazebo/Unity سیمولیشن، اور Vision-Language-Action ماڈلز کو استعمال کرتے ہوئے جسمانی دنیا میں سمجھ، منطق، اور کارروائی کر سکتے ہیں۔

## ڈھانچہ

کورس چار بنیادی ماڈیولز میں ترتیب دیا گیا ہے:

- **ہفتہ 3**: [ROS 2 کی افادیت](./docs/ros2/week3.md)

### ماڈیول 2: Gazebo/Unity - ڈیجی ٹل ٹوئنز (ہفتے 4-6)
- **ہفتہ 4-5**: سیمولیشن ماحول اور Gazebo انضمام
- **ہفتہ 6**: یونٹی انضمام اور اعلیٰ سینسر
- **ہفتہ 4-5**: [Gazebo کی بنیاد اور ماحول کی تیاری](./docs/gazebo-unity/week4-5.md)
- **ہفتہ 6**: [Unity انضمام اور اعلیٰ سینسر](./docs/gazebo-unity/week6.md)

- **ہفتہ 13**: کورس کا اختتام اور کیپسٹون پروجیکٹ

## تکمیل کے معیار

اس کورس کے اختتام پر، آپ کو درج ذیل کر سکنے کی صلاحیت ہوگی:

- ROS 2 کے استعمال سے روبوٹکس سسٹم ڈیزائن اور نافذ کرنا
- ڈیجی ٹل انٹیلی جنس اور جسمانی اجسام کے درمیان انضمام کو سمجھنا
- جی بہترین سیمولیشن اور ہارڈویئر پلیٹ فارمز کے ساتھ کام کرنا
- روبوٹکس کاموں کے لیے Vision-Language-Action ماڈلز نافذ کرنا
- پیچیدہ روبوٹکس مشنز کو منصوبہ بندی اور انجام دینا

## تنصیب

کتاب کو مقامی طور پر چلانے کے لیے:

```bash
npm install
npm run start
```

یا اگر yarn استعمال کر رہے ہیں:

```bash
yarn install
yarn start
```

کتاب کو تیار کرنے کے لیے:

```bash
npm run build
```
